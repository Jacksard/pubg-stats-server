# pubg-stats-server
Node/Express/Axios for PUBG API calls

How to run:
1. create .env file in the root folder.
2. define two global variables (https://www.npmjs.com/package/dotenv)

API_KEY = Obtained from - https://developer.pubg.com/
MONGODB = MongoDB connection link with your credentials - https://www.mongodb.com/ 

3. npm install
4. npm run server
